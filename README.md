# Aligoo Starter Kit

Aligoo Starter Kit is a template for creating a new project with next.js and Sanity. It uses pnpm as a package manager and has a monorepo structure with a studio and web folder.

## Usage 
Create a project on sanity.io and copy project id and dataset from the project settings.
Add them to the .env.local file in the web folder.   
Run pnpm run dev 
to start the development server.
